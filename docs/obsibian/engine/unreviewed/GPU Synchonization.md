В современых Render API  программист должен сам отвечать за создание и удаление ресурсов [[Типы ресурсов]]. По этому CPU и GPU должны быть синхронизероваы между собой. 

Одним из возможных способов синхронизаци может быть использование [[Fence]].  В конце каждой очереди команд отправляемоей на GPU записывется команда об отправки сигнала об измении fence, CPU поток на следуюшем кадре будет ждать этого сигнала.
